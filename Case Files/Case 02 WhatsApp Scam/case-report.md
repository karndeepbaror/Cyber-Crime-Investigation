# Case Report  
**Case 02: WhatsApp Financial Scam**

**Case Category**
Social Engineering & Financial Fraud

**Investigation Type:**
Simulated Cyber Crime Investigation (Educational & Ethical)

**Date of Documentation**
December 2025

**Investigator:**
Karndeep Baror  
`Digital Forensics & Cyber Crime Investigation`  (Portfolio Case)


**1. Case Background:**

A user reported receiving suspicious messages on WhatsApp claiming to be from a banking service.  
The messages requested personal information and prompted links to phishing pages.

No real money loss was involved; this is a simulated investigation for portfolio purposes.



**2. Scope of Investigation:**

- Passive OSINT analysis
- Review of message patterns and sender profiles
- No interaction with scam links or accounts
- Ethical, legal investigation only



**3. Investigation Actions Taken:**

- Analysis of message headers (simulated metadata)  
- Phone number verification  
- Pattern identification for common WhatsApp scams  
- Timeline reconstruction of scam messages


**4. Findings:**

- Sender numbers were newly registered  
- Messages contained typical scam keywords  
- Links were phishing URLs (simulated)  
- Targeted users were randomly selected  



**5. Conclusion:**

The case illustrates the common structure of `WhatsApp scams` targeting financial data.  
Early detection and awareness are critical.

Status: `CLOSED`
