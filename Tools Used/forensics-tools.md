# Digital Forensics Tools & Techniques

This document outlines commonly used tools and techniques
for `digital forensic analysis` in `cyber crime investigations.`

***Disk & File System Analysis:***

- **Autopsy**
  Open-source digital forensics platform for disk analysis.
- **The Sleuth Kit**
  File system examination and timeline creation.

***Memory & Volatile Data Awareness:***

- **Volatility Framework**
  Memory artifact understanding `(theoretical knowledge).`
- **Process & Network Artifact Review**
  Awareness of RAM-based indicators.

***Log & Artifact Analysis:***

- **System Log Review**
  Authentication, service, and access logs.
- **Application Logs**
  Browser, email client, and messaging app artifacts.

***Evidence Handling Standards:***

- Read-only analysis
- Evidence hashing awareness
- Chain of custody documentation
- No modification of original data

***Compliance Followed:***

- Indian IT Act
- Standard `DFIR` principles
- Court-acceptable reporting prDFIR
