# Network Analysis Tools

This document highlights tools and methods used
to understand `network behavior` during investigations.

***Traffic Analysis:***

- **Wireshark**
  Packet inspection for protocol understanding.
- **TCP/IP Stack Analysis**
  Connection behavior and anomaly awareness.

***Port & Service Awareness:***

- **Nmap (Knowledge Level)**
  Understanding service exposure concepts.
- **Service Enumeration Theory**
  Risk identification without exploitation.

***Email & Phishing Analysis:***

- **Email Header Analysis**
  SPF, DKIM, DMARC interpretation.
- **Mail Transfer Path Review**
  Identifying spoofed sender infrastructure.

***Network Investigation Ethics:***

- No packet injection
- No scanning of `unauthorized systems`
- Passive inspection only
- Legal authorization required for `live analysis`
